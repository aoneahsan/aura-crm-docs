# Changelog

Release notes for AuraCRM, latest first.

## v1.0.0 — Initial public release

The first public release of AuraCRM, the ambient, voice-first client operations engine.

**Features**

- Voice capture (foreground, tap-to-record) and manual notes, with transcription via OpenAI Whisper.
- AI orchestration that turns a session into reviewable proposals — engagement-stage change,
  style-matched follow-up email, professional tasks, client homework, and overage invoices — with
  PII redacted before the language model and **nothing sent or charged without your approval**.
- Clients with a finite-state engagement pipeline (Discovery → Proposal → Active Engagement →
  Renewal Review) and an audited history.
- Tasks split into your professional to-dos and client homework.
- Overage invoices drafted for approval; direct bank-transfer billing.
- Private per-client progress portals with magic-link access, homework check-off, and revocation.
- Admin panel: users and roles, subscription approvals, waitlist export, analytics, an audit log,
  and short links.
- Google sign-in (web and mobile), theme customiser with cross-device sync, and optional push
  notifications.
- Data on Supabase Postgres with row-level security; files and email via FilesHub; GDPR data export
  and account deletion; HIPAA-ready architecture.
